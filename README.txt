'''Данный проект создан для того, чтобы проверить насколько
сверточная нейросеть YOLO точно определяет положение объектов, а также
выяснить влияение количества объектов в кадре на эту точность'''

#исходники фотографий и видео можно скачать из google disk

Алгоритм действий:
1)берем исходник нашего  видео , разбиваем его на изображения.
Для минимизации затрат на обработку данных в дальнейшем из видео рекомендуется брать каждый 5 кадр
2)размечаем изображения с помощью библиотеки labelimg (берем за эталонное нахождение объекта)
3)обрабатываем изображения с помощью нейросети YOLO
4)находим отклонение от идеала для каждого кадра . В нашем случае берется не отклонение от каждой
из оси , а вектор соединяющий эталонную разметку и разметку нейронной сети
5)визуализируем данные с помощью библиотеки matplotlib
